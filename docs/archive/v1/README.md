# v1 Historical Archive

This directory contains historical planning material from the earlier `AI Theater` / `AITube` direction.

These documents are preserved for context only.
They are not the current `vynema` implementation contract unless a current baseline document explicitly adopts a specific part.

Current baseline documents:

- `docs/requirements/vynema-mvp-requirements.md`
- `docs/architecture/vynema-architecture.md`
- `PROJECT-STATUS.md`

Known differences from the current direction:

- v1 assumed a more YouTube-like product framing.
- v1 included paid-service assumptions that are not accepted for the current MVP baseline.
- v1 included historical Claude/team workflow references that are no longer active contributor instructions.
- v1 design assets and detailed specs may still be useful as references, but must be reviewed before adoption.
